# MemPalace Bot 🤖

Telegram-бот, который общается с искусственным интеллектом, ищет информацию в ваших личных заметках (MemPalace), скачивает видео с YouTube, делает транскрипции и многое другое.

## 🔧 Что нужно для установки

- **Компьютер Mac** (Intel или Apple Silicon M1/M2/M3)
- **Доступ в интернет**
- **Аккаунт Telegram**
- **Умение копировать и вставлять текст** — всё остальное сделают инструкции ниже

---

## 📦 Пошаговая установка

### Шаг 1. Установите Homebrew (если ещё нет)

Откройте программу **Терминал** (нажмите `⌘ + Пробел`, введите `Терминал` и нажмите Enter).

Скопируйте и вставьте эту команду в Терминал, нажмите Enter:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Если спросит пароль — введите пароль от вашего Mac (символы не будут видны, это нормально). Процесс может занять несколько минут.

### Шаг 2. Установите Python и FFmpeg

Вставьте в Терминал:

```bash
brew install python@3.12 ffmpeg
```

### Шаг 3. Установите MemPalace

```bash
brew install anomalyco/tap/mempalace
```

### Шаг 4. Скачайте бота

Откройте папку, в которую хотите установить бота. Например:

```bash
cd ~/Documents
git clone https://github.com/ВАШ_ЛОГИН/mempalace_bot.git
cd mempalace_bot
```

Если у вас нет ссылки на GitHub — попросите у автора архив или вставьте сюда свой репозиторий.

### Шаг 5. Создайте виртуальное окружение

```bash
python3.12 -m venv venv
source venv/bin/activate
```

### Шаг 6. Установите зависимости

```bash
pip install -r requirements.txt
```

### Шаг 7. Создайте файл с ключами

Скопируйте образец:

```bash
cp .env.example .env
```

Откройте файл `.env` в любом текстовом редакторе и заполните:

```
TELEGRAM_BOT_TOKEN= — ваш токен от @BotFather (как получить — ниже)
ADMIN_ID= — ваш ID в Telegram (как узнать — ниже)
GEMINI_API_KEY= — ключ от Google Gemini (как получить — ниже)
```

**Как получить TELEGRAM_BOT_TOKEN:**
1. Откройте Telegram, найдите @BotFather
2. Напишите `/newbot`, придумайте имя (например, MyMemBot)
3. BotFather пришлёт токен — скопируйте его (выглядит как `123456:ABCdef...`)
4. Вставьте вместо `your_telegram_bot_token`

**Как узнать свой ADMIN_ID:**
1. Откройте Telegram, найдите @userinfobot
2. Нажмите /start — бот покажет ваш ID (число, например 123456789)
3. Вставьте это число вместо 0 в строку ADMIN_ID

**Как получить GEMINI_API_KEY:**
1. Откройте site: https://aistudio.google.com/apikey
2. Нажмите «Create API key»
3. Скопируйте ключ и вставьте в .env

### Шаг 8. Запустите бота

```bash
python3 main.py
```

Если всё сделано правильно, вы увидите:
```
🦾 MemPalace запущен.
```

Теперь откройте Telegram, найдите своего бота и нажмите /start.

## 🎮 Как пользоваться

После запуска бот покажет кнопки:

| Кнопка | Что делает |
|--------|-----------|
| 🆕 Новый диалог | Создаёт новый чат с ИИ |
| 📂 Список чатов | Показывает ваши диалоги |
| ⚙️ Настройки | Голос, модель ИИ |
| 🔍 Поиск по крылу | Ищет в ваших заметках |
| 🔄 Синхронизация | Сохраняет диалог в MemPalace |
| 📹 Скачать видео | Скачивает YouTube-ролик (480/720p) |
| 🎵 Скачать MP3 | Скачивает аудио с YouTube |

Также можно просто писать боту сообщения — он ответит через ИИ.

## 🧹 Если что-то пошло не так

**Бот не отвечает:**
- Проверьте, запущен ли скрипт (в Терминале должно быть видно `Bot polling started`)
- Проверьте, правильно ли введён токен в `.env`
- Напишите мне в Telegram: @author_username

**Ошибка «Module not found»:**
- Выполните `pip install -r requirements.txt` ещё раз
